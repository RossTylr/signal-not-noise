# 00 — Orientation

## How to Use This Repo

This repository teaches dimensionality reduction using the **Feynman Technique**: learn something, then explain it in plain language. If you can't explain it simply, go back and learn it again.

### The Structure

Each notebook follows a consistent pattern:

1. **The One-Sentence Version** — The entire concept in one sentence
2. **The Story** — Plain language walkthrough, no code, no maths
3. **Interactive Code** — Build it, see it, break it
4. **The Maths Behind This** — Collapsible, optional, for when you're ready
5. **Where This Matters** — Connection to real-world problems
6. **Feynman Check** — Can you actually explain this now?

### Rules for Yourself

- Don't skip the Feynman Checks. They're where learning happens.
- If a visual confuses you, change the parameters and re-run it.
- Build from scratch before importing from sklearn.
- If you can explain it to a non-technical colleague, you understand it.
- If you can't, you've found exactly what to study next.

### Prerequisites

- Basic Python (numpy, matplotlib)
- Comfort with arrays and basic statistics (mean, variance)
- No linear algebra required upfront — we'll build what we need
